ChatKit UI is customized by passing an options object. The shape is the same in React (`useChatKit({ ... })`) and the web component (`chatkit.setOptions({ ... })`).

You can visit https://chatkit.studio/playground to experiment with different customization options and copy the options object.

<Tabs syncKey="language">
<TabItem label="React">

```jsx
const { control } = useChatKit({
  theme: { colorScheme: "dark", radius: "round" },
  header: { enabled: true },
  history: { enabled: true },
  startScreen: {
    greeting: "Welcome to FeedbackBot!",
    prompts: [{ label: "Report a bug", prompt: "Report a bug", icon: "bolt" }],
  },
  composer: {
    placeholder: "Type your product feedback…",
    tools: [{ id: "rate", label: "Rate", icon: "star", pinned: true }],
  },
});
```

</TabItem>
<TabItem label="Web component">

```js
chatkit.setOptions({
  theme: { colorScheme: "dark", radius: "round" },
  header: { enabled: true },
  history: { enabled: true },
  startScreen: {
    greeting: "Welcome to FeedbackBot!",
    prompts: [{ label: "Report a bug", prompt: "Report a bug", icon: "bolt" }],
  },
  composer: {
    placeholder: "Type your product feedback…",
    tools: [{ id: "rate", label: "Rate", icon: "star", pinned: true }],
  },
});
```

</TabItem>

## Pick a customization

- [Theme](./customize-theme): Adjust color scheme, accent palette, rounding, density, and typography.
- [Header and history](./customize-header-history): Toggle the header, add custom actions, and configure history controls.
- [Start screen](./customize-start-screen): Show a greeting and suggested prompts when a new thread opens.
- [Composer](./customize-composer): Guide inputs with placeholder text, attachments, tools, model picker, and entity mentions.
- [Thread item actions](./customize-thread-item-actions): Enable or disable feedback and retry buttons on assistant messages.
